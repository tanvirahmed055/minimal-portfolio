Expense Manager

Expense Manager is a Full Stack MERN Application where we can add transactions such as income or expense and that's get added as transaction history. The app also shows the real time current balance and all of the previous transactions that has been made before. User can also delete any transaction from the transaction history.


The front-end of the application is built with React JS. The back-end of the application is made with Node & Express JS and MongoDB as database so that the any transaction made gets stored and get fetched from the back-end.
